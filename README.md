# ★ NOVA STRIKE

Jogo de nave espacial 2D — moderno, estiloso, sem erros.
Feito 100% com HTML5 Canvas, CSS3 e JavaScript puro.

## 🎮 Controles

| Tecla | Ação |
|-------|------|
| `←` `→` | Mover nave |
| `ESPAÇO` | Atirar |
| `P` | Pausar / Continuar |

## ✨ Funcionalidades

- 🚀 Nave com visual neon e chama no motor
- 👾 3 tipos de inimigos: Normal, Tank e Boss
- 🎯 Inimigos atiram de volta (mira no jogador!)
- 💙 Sistema de vidas com invencibilidade temporária
- 🌟 100 estrelas animadas no fundo
- 🔊 Sons gerados por Web Audio API
- ⏸️ Tela de pausa
- 📈 Level up automático a cada 300 pontos
- 🏆 Pontuação e recorde

---

## 🛠️ Tecnologias

- HTML5 Canvas, CSS3, JavaScript ES6+
- Web Audio API (sem arquivos de som externos)
- Google Fonts: Exo 2 + Share Tech Mono
